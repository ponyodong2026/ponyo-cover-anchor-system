# 四套封面模板完整参数表

---

## 🔴 模板一：冲突型

**核心公式：** 反直觉标题 × 强对比色 × 人物遮罩

**适用场景：** 知识博主、AI工具介绍、观点输出、方法论内容

### 结构参数

| 元素 | 规格 |
|------|------|
| 锚点位置 | 人脸居中偏下 + 标题在上1/3 |
| 锚点逻辑 | 人脸引导注意力 → 视线上移至标题 |
| 主标题位 | 上1/3，左对齐或居中 |
| 人物遮罩 | 占画面50-70%，底部出血 |
| 背景分区 | 上2/3强色背景 + 下1/3人物区 |

### 配色规则

| 配色组 | 主色 | 撞色 | 底色 | 情绪 |
|--------|------|------|------|------|
| 权威红 | #C0392B | #F7F7F7 | #1A1A1A | 紧迫感、专业 |
| 科技蓝 | #2471A3 | #F1C40F | #0D1B2A | 理性、前沿 |
| 警示橙 | #E67E22 | #FFFFFF | #2C2C2C | 警示、干货 |
| 高级黑 | #1A1A1A | #E74C3C | #F5F5F0 | 极简、高端 |

### 字重层级

- 主标题：粗体（Bold）/ 字号比副标题大2-3倍
- 副标题：中等（Medium）/ 颜色用主色撞色
- 角标/标签：细体（Light）/ 小字带色块背景

### 对比度阈值

主标题与背景对比度 ≥ 4.5:1（缩略图可读标准）

## 🟡 模板二：数字型

**核心公式：** 大号数字锚点 × 色块分割 × 悬念副标题

**适用场景：** 干货攻略、步骤教程、成果展示、工具测评

### 结构参数

| 元素 | 规格 |
|------|------|
| 锚点位置 | 中心大号数字（数字即锚点）|
| 数字字号 | 占画面高度的30-40% |
| 色块分割 | 横向或斜向分割，2色或3色 |
| 悬念副标题 | 数字下方或上方，留出「为什么」的悬念 |
| 留白 | 数字周围保留呼吸感，不要塞满 |

### 配色规则

| 配色组 | 主色 | 撞色 | 底色 | 适用内容 |
|--------|------|------|------|---------|
| 珊瑚活力 | #E74C3C | #FFFFFF | #FDF6EC | 生活干货 |
| 薄荷清新 | #1ABC9C | #2C3E50 | #F0FFFE | 健康/技巧 |
| 深蓝学术 | #154360 | #F1C40F | #FDFEFE | 知识/学习 |
| 暖橙收益 | #F39C12 | #1A1A1A | #FFFBF0 | 变现/收益 |

### 字重层级

- 数字：超粗体（Black/Heavy）/ 占满版面
- 单位词（天/个/步/万）：中等，数字旁小字
- 副标题：细体，颜色弱化，制造层次

### 悬念副标题公式

```
数字 + [做什么] + [结果/转变] + ？
例：7天 + 学会用AI做视频 + 真的可以吗？
例：3个方法 + 让封面点击率翻倍
```

## 🟢 模板三：截图型

**核心公式：** 真实截图遮罩 × 手写感标注 × 低饱和底色

**适用场景：** 案例分享、工具测评、结果展示、对话截图

### 结构参数

| 元素 | 规格 |
|------|------|
| 锚点位置 | 截图本身就是锚点（真实感=可信度）|
| 截图位置 | 居中，略向下，上方留标题位 |
| 截图角度 | 轻微旋转2-5度，增加随意感 |
| 标注元素 | 手写圆圈、箭头、下划线（增加注意力引导）|
| 底色选择 | 低饱和、不抢截图风头 |

### 配色规则

| 配色组 | 底色 | 标注色 | 标题色 | 适用内容 |
|--------|------|--------|--------|---------|
| 暖米纸感 | #F5F0E8 | #E74C3C | #1A1A1A | 生活攻略 |
| 冷白简约 | #F8F9FA | #2471A3 | #1A1A1A | 工具测评 |
| 马卡龙粉 | #FDE8F0 | #C0392B | #2C2C2C | 情感/生活 |
| 深色沉浸 | #1A1A2E | #F1C40F | #FFFFFF | AI/科技 |

### 字重层级

- 主标题：粗体，上方居中，色彩醒目
- 截图内文字：保留原样（真实感来源）
- 手写标注：保持随意感，不要太工整

### 真实感要素（重要）

截图型成功的关键是「真实感」，避免以下失误：
- ❌ 截图太完美（要有轻微阴影、边缘感）
- ❌ 底色太鲜艳（会抢截图的视觉权重）
- ❌ 标注太工整（要像手写、手画的）
- ✅ 截图轻微投影、轻微旋转
- ✅ 标注元素用红色或荧光色

## 🔵 模板四：情绪型

**核心公式：** 全出血人物图 × 单行大字 × 撞色条

**适用场景：** 情感类、生活类、Vlog、故事内容

### 结构参数

| 元素 | 规格 |
|------|------|
| 锚点位置 | 人脸（占画面40-60%）|
| 人物位置 | 居中或下2/3，上方留文字区 |
| 文字区 | 上1/3叠加半透明色条或直接在图上压字 |
| 撞色条 | 水平色条，高度约15-20%的画面高，置于上方或下方 |
| 字数 | 控制在8字以内，字越少越有力 |

### 配色规则

| 配色组 | 撞色条 | 字色 | 配图色调 | 情绪 |
|--------|--------|------|---------|------|
| 激情红 | #C0392B | #FFFFFF | 暖色调人物 | 愤怒/励志 |
| 忧郁蓝 | #154360 | #FFFFFF | 冷色调人物 | 思考/治愈 |
| 阳光橙 | #F39C12 | #1A1A1A | 暖阳人物 | 积极/温暖 |
| 高级灰 | #2C2C2C | #FFFFFF | 任意人物 | 优雅/高端 |
| 纯黑沉浸 | 无色条，直接压字 | #FFFFFF | 任意 | 电影感 |

### 字重层级

- 主标题：超粗体，字号极大，占满撞色条宽度
- 字数：≤8字（超过就改模板）
- 辅助信息：小字角标，不抢主标题

### 情绪型文案公式

```
[状态] + [转折] 或 [感受]
例：不想上班，但很自由
例：被说太敏感，然后呢
例：一个人也很好
```

## 通用锚点规则

### 人眼扫描路径
- Z轴扫描：左上 → 右上 → 左下 → 右下
- F轴扫描：上方横向 → 左边纵向
- **结论：** 最强位置是左上和上方水平中心

### 缩略图可读性测试
在做完封面后，缩小到手机屏幕宽度的1/4大小（约80px宽），仍能看清主标题 = 合格

### 色彩对比度最低标准
| 使用场景 | 最低对比度 |
|---------|----------|
| 主标题 vs 背景 | 4.5:1 |
| 副标题 vs 背景 | 3:1 |
| 缩略图模式 | 建议7:1以上 |
